# E-Commerce Product Data Analysis and Predictive Modeling

This project analyzes e-commerce product data using data science techniques. 
It includes web scraping, data cleaning, statistical analysis, MySQL storage, 
machine learning models, hyperparameter tuning, and Power BI dashboard visualization.

## Tools & Technologies
- Python (pandas, sklearn, matplotlib)
- MySQL
- Power BI
- Jupyter Notebook

## Project Workflow
1. Web Scraping
2. Data Cleaning
3. Statistical Analysis
4. Supervised & Unsupervised Learning
5. Hyperparameter Tuning
6. Dashboard Visualization
